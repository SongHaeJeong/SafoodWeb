## SAFOOD (SAFE + FOOD)

__개요__ : 현대인의 과영양 상태를 체크하고 개인별 알레르기 식품들을 관리할 목적으로, 식품 별 첨가물(원재료)과 식품 별 영양 성분을 관리하는 웹 서비스
